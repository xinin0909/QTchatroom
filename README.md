# QTchatroom
a easy small chat Tools  for everyone(use PYQT) 
项目分为两部分
# 1，客户端
  使用PYQT制作界面，界面布局仿照微信电脑版，可以支持不同局域网连接，使用清爽，与服务端连接时使用信号与槽机制保证连接的畅通性
  可以在/image/head/目录下添加自己喜欢的表情包，更具有人性化
  详情请见文件夹 可执行程序打包/client
# 2，服务端
  使用线程池保证多人连接时的通讯畅通，系统稳定，支持在多环境下进行部署
